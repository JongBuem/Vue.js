# Vue

홈페이지주소 : [https://jongbuem.github.io/vue.js/](https://jongbuem.github.io/vue.js/)

## **지시 업무**

- 작성된 유튜브 페이지 vue.js로 다시작성
- 조건부 렌더링사용 [ v-if, v-for, v-show ]
- 바인딩사용 [ v-bind or : ]
- 템플릿 문법 [ 속성, 디렉티브, 동적 전달인자, 약어, 수식어 ]
- vue 인스턴스는 하나로만 작성
- 주석은 소스옆에 붙이거나 위줄에 작성
